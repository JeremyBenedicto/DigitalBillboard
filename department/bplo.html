üì¢ 4. Feedback at Complaints Mechanism

Paano Magbigay ng Feedback
  • Sagutan ang Client Feedback Form
  • Ihulog sa drop box ng Public Assistance and Complaint Desk
  • O mag-email sa: gabaldonbplo@gmail.com/gabnecijahrmo@gmail.com
Processing ng Feedback
  • Tuwing Biyernes: Bubuksan ng staff ang drop box
  • Mga feedback ay irereply within 3 days
Paano Mag-file ng Complaint
  • Sagutan ang Client Complaint Form
  • Ihulog sa drop box sa harap ng Public Assistance and Complaint Desk
  • Pwede ring mag-text, tumawag, o mag-email (siguraduhing ilagay ang mga sumusunod):
    • Pangalan ng siniriklamo
    • Incident details
    • Evidence (kung meron)
Complaint Processing
  • Araw-araw: Ini-check ng Municipal Administrator ang drop box
  • Mag-iimbestiga at magbibigay ng report sa Mayor
  • Irereply sa client ang resulta
